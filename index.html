Setup

Save these files in one folder and open index.html in a browser, or upload the folder to any static host.

  1. GitHub Pages: create a GitHub repo, upload index.html, styles.css, and script.js, then enable Pages from the repo settings.
  2. Netlify, Vercel, or Render: create a new static site and upload or import the folder. No build command is needed.
  3. From an iPhone: use the GitHub mobile app or github.com in Safari to upload the files, then enable GitHub Pages from the repository settings page.
